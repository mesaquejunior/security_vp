Problemas encontrados:

Os arquivos baixados, ao serem abertos pelo excel, exibem mensagem de que o arquivo está corrompido. Na realidade o arquivo não é um xls, é um txt separado por tab. Por esta razão foi escrito um script em powershell para renomear os arquivos para csv e posteriormente converter os dados para xlsx.
Foi necessário padronizar o formato de todos os campos para unificar os dados para importação em tabela única
Há tipos de informações em campos errados. Ex. String em campo que precisa ser float. A razão é que o operador preenche a informação em campo errado. Ex. Nome de rua no lugar de número. UF no lugar de LATITUDE
Há dados com campos em ordem diferente do padrão
